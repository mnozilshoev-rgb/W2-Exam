# 🚀 INNOMA.VC Landing Page

A modern and responsive landing page for **INNOMA.VC**, an international analytics and accelerator platform helping IT startups launch and scale in global markets.

This project was built by recreating a professional Figma design using **HTML**, **CSS**, and **Tailwind CSS** with a focus on pixel-perfect implementation, responsive layout, and clean UI structure.

---

## 🌍 Project Overview

INNOMA.VC helps startups and IT businesses expand into international markets including:

* Middle East (MENA)
* Asia
* Latin America
* Africa

The website showcases:

* Company services
* Accelerator program
* Market analytics
* Pricing plans
* Team & experts
* Contact section
